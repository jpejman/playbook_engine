# Continuous Pipeline v0.3.1 — Evaluation Commands Guide  
**Version:** v0.3.1  
**Timestamp (UTC):** 2026-04-22  

---

## 1. Basic Single-Model Evaluation

```bash
python -m scripts.prod.continuous_pipeline_v0_3_1.evaluate_models_v0_3_1 \
  --models gemma3:4b \
  --cve-list CVE-2023-4863 \
  --eval-label "v3_1_strict_pass_test" \
  --execution sequential

2. Multi-Model Comparison
python -m scripts.prod.continuous_pipeline_v0_3_1.evaluate_models_v0_3_1 \
  --models gemma3:4b,qwen2.5-Coder,qwen3.5:4b \
  --cve-list CVE-2023-4863,CVE-2024-12345 \
  --eval-label "v3_1_multi_model_test" \
  --execution sequential

3. Batch Evaluation from Queue
python -m scripts.prod.continuous_pipeline_v0_3_1.evaluate_models_v0_3_1 \
  --models gemma3:4b \
  --cve-source queue \
  --limit 5 \
  --eval-label "v3_1_queue_test" \
  --execution sequential

4. Parallel Execution (Faster)
python -m scripts.prod.continuous_pipeline_v0_3_1.evaluate_models_v0_3_1 \
  --models gemma3:4b,qwen3.5:4b \
  --cve-source opensearch \
  --limit 10 \
  --eval-label "v3_1_parallel_test" \
  --execution parallel \
  --max-workers 2

5. With Timeout Control
python -m scripts.prod.continuous_pipeline_v0_3_1.evaluate_models_v0_3_1 \
  --models gemma3:4b \
  --cve-list CVE-2023-4863 \
  --eval-label "v3_1_timeout_test" \
  --timeout 120 \
  --execution sequential

6. Force Re-evaluation
python -m scripts.prod.continuous_pipeline_v0_3_1.evaluate_models_v0_3_1 \
  --models gemma3:4b \
  --cve-list CVE-2023-4863 \
  --eval-label "v3_1_force_test" \
  --force \
  --execution sequential

7. From CVE File
# Create input file
echo "CVE-2023-4863" > test_cves.txt
echo "CVE-2024-12345" >> test_cves.txt
python -m scripts.prod.continuous_pipeline_v0_3_1.evaluate_models_v0_3_1 \
  --models gemma3:4b \
  --cve-source file \
  --cve-file test_cves.txt \
  --eval-label "v3_1_file_test" \
  --execution sequential

8. Diagnostics and Schema Check
python -m scripts.prod.continuous_pipeline_v0_3_1.diagnostics_v0_3_1

Alternative direct execution:

python scripts/prod/continuous_pipeline_v0_3_1/diagnostics_v0_3_1.py

9. Test Validation Modules
